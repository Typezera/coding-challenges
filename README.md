# Coding Challenges

Este repositório reúne minhas soluções para desafios de programação resolvidos em diferentes plataformas. O objetivo é praticar lógica de programação, estruturas de dados, algoritmos e conhecer melhor diferentes linguagens ao longo da minha jornada de estudos.

## Plataformas

- Beecrowd *(em andamento)*
- Futuramente: LeetCode
- Futuramente: Codeforces

## Linguagens

- Go (Golang)

## Estrutura

```text
coding-challenges/
├── beecrowd/
│   └── beginner/
│       ├── 1000-hello-world/
│       ├── 1001-extremely-basic/
│       └── ...
├── leetcode/
└── codeforces/
```

## Objetivos

- Praticar resolução de problemas.
- Melhorar lógica de programação.
- Aprender novas linguagens.
- Evoluir em algoritmos e estruturas de dados.
- Manter um histórico público da minha evolução.
